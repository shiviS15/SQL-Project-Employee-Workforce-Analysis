# SQL-Project-Employee-Workforce-Analysis

SQL Project: Employee Workforce Analysis

I'm excited to share my latest SQL portfolio project focused on Employee Workforce Analysis. This project demonstrates how SQL can be used to clean, explore, and analyze employee data to generate meaningful business insights.

Project Highlights:
📊 Performed data cleaning and validation to identify missing values and duplicates.
📈 Conducted exploratory data analysis (EDA) to understand workforce distribution across departments, cities, and employment status.
💰 Analyzed salary trends, bonus distribution, and department-wise payroll.
⭐ Evaluated employee performance ratings and experience levels.
📅 Explored hiring trends using date functions.
🔍 Solved real business problems using aggregate functions, subqueries, CTEs, and window functions such as RANK(), DENSE_RANK(), and ROW_NUMBER().
